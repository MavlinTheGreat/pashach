Пашач 1.0 - первая итерация веб-познания

Пашач 1.0 - учебный сайт-имиджборда, разработанная Никитином Павлом в целях освоения базовых концептов веб-разработки с использованием Java Spring, PostgreSQL, Bootstrap.

Всего планируется 3 версии имиджборды Пашач.

Пашач 1.0 - Базовая функциональность

В этой версии Пашача будет разработана его базовая функциональность, такая как:

  • Разбиение сайта на доски;
  • Создание тредов;
  • Постинг сообщений с возможностью подписываться;
  • Прикладывание картинок (основная учебная задача разработки);
  • Доступ из всемирной телекомунникационной сети "Интернет", обеспеченный использованием ПК в качестве сервера.

Таким образом, первая версия разработки позволит доосвоить и закрепить базовые навыки разработки веб-приложений на Java Spring, подробнее углубиться в вопрос хранения медиа-ресурсов на сервере, разобраться с основами настройки персонального сервера.

Пашач 2.0 - Прикрытие вольницы

В этой версии Пашача будет разработан его функционал, закрытый от глаз рядового пользователя, такой как

  • Скрытая система аккаунтов с использованием Spring Security;
  • Панель администрирования - создание и удаление досок, менеджмент банов, состава пользователей, закреплёных тредов;
  • Инструменты модерации сайта - удаление и изменение постов, блокировка по IP, архивация тредов и запрет на ответ в них;
  • Внедрение базовых средств аналитики.

Таким образом, вторая версия разработки позволит на практике освоить создание механизмов аутентификации с помощью Spring Security и разработать средства опосредованного управления сайтом без допуска к его исходному коду.

Пашач 3.0 - Глобальная безопасность

В последней предполагаемой версии Пашача будут произведены, главным образом, нефункциональные, но архитектурные доработки сайта. Планируется:

  • Переработка архитектуры баз данных под большие объёмы;
  • Увеличение безопасности сайта, защиты от взлома сайта;
  • Размещение сайта на внешнем, возможно облачном сервере с соответствующей настройкой;
  • Приобретение домена во всемирной телекоммуникационной сети "Интернет" и его присвоение сайту.

Таким обрахом, третья версия имиджборды Пашач позволит ей являться полноценным, с технической точки зрения, продуктом, способным реализовывать базовый функционал имиджборд.
